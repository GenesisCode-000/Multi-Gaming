# PROJECT MULTI | Dashboard 🚀

**PROJECT MULTI** is a modern, fast, and sleek dashboard interface designed to centralize gaming tools, Roblox scripts, and HTML game packs.



## 🛠️ Core Features

* **Modern Design & Dark Theme:** A dark, minimalist interface with "glassmorphism" effects for a premium look.
* **Tabbed Navigation:** An intuitive structure to organize scripts, games, and system tools efficiently.
* **Fluid Interaction:** Custom "Toast" notifications for smooth clipboard copying without interrupting the user experience.
* **Responsive:** Fully optimized for all screen sizes (Desktop, Tablet, and Mobile).
* **Integrated Game Viewer:** Launch HTML games directly within a modal window without leaving the dashboard.

## 💻 Tech Stack

* **HTML5**: Content structure and semantics.
* **CSS3**: Styling, animations, and layout (external CSS file).
* **JavaScript**: Navigation logic, clipboard management, and modal control.

## 🚀 Installation & Usage

1. Clone the repository to your local machine or download the ZIP file.
2. Ensure that the `index.html` and `style.css` files are in the **same directory**.
3. Open `index.html` in your preferred web browser.
